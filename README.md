# ClipStudy 1.0

用 YouTube 视频片段学英语的 macOS 工具：下载带字幕的视频，按句听写、AB 循环、倍速精听。
*A macOS app to learn English with YouTube clips: download subtitled videos, dictate sentence-by-sentence, A-B loop, and slow-motion listening.*

## ✨ 功能 / Features
- 📥 **一键下载** YouTube 视频（自动合并音视频、抓取字幕），内置 yt-dlp 与 FFmpeg，**无需自行安装任何依赖**
- ✍️ **听写模式**：按标点自动断成完整句子，一句循环播放，可显示/隐藏答案，逐句推进
- 🔁 **AB 循环**：任意片段反复精听
- 🐢 **倍速播放**：0.5×–2×，慢速练听力
- 📝 **字幕**：实时显示，可隐藏
- 🖥️ **全屏 + 可调列表栏**
- 🌐 **中英文界面**（跟随系统，可手动切换）

## 💻 系统要求 / Requirements
- macOS（Apple Silicon / Intel）

## 📦 安装 / Install
1. 下载下方 `ClipStudy.dmg`
2. 双击打开，把 **ClipStudy** 拖到 **Applications**
3. 启动即可（已签名 + 公证，双击直接打开，无安全警告）
